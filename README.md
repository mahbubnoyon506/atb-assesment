## Getting Started

First, install the necessary packages:

````bash
npm install
#or
npm i
# or
yarn install


Now, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
````

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
